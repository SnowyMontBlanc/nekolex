# 014: プロフィール画面

## 概要
ユーザーのレベル、XP、発見品種数、クイズ正解率などの統計情報を表示するプロフィール画面を実装する。

## 依存先
003 (ナビゲーション), 005 (UserContext)

## ブロック対象
なし

### 2. レベル & XP 表示
- 現在のレベルとタイトル
- XP プログレスバー (現在値 / 次レベル必要値)
- xp-calculator から計算

### 3. 統計情報
UserContext から取得:
- 発見品種数 / 全品種数
- クイズ実施回数
- 全体正解率 (total_correct_answers / total_questions)
- 学習日数 (最初の発見日からの日数)

### 4. 実績セクション (将来実装用スタブ)
- ロック表示のみ (MVP ではアンロック不可)
- バッジ名と条件の表示

### 5. 設定セクション
- サウンド ON/OFF トグル
- 触覚フィードバック ON/OFF トグル
- データリセット (確認ダイアログ付き)

## 完了条件
- レベルと XP が正しく表示される
- XP プログレスバーが正しい割合を示す
- 統計情報が UserContext のデータと一致する
- 設定の変更が保存される
- データリセットが確認後に実行される
