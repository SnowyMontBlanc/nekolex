# 012: クイズ画面実装

## 概要
難易度選択、4択クイズ出題、正解/不正解判定を行うクイズ画面を実装する。

## 依存先
003 (ナビゲーション), 005 (UserContext), 011 (クイズ生成ロジック)

## ブロック対象
013, 016

## 作業内容

### 1. クイズタブ画面 (`app/(tabs)/quiz.tsx`)

#### 難易度選択 UI
```
┌─────────────────────────────┐
│  🎯 復習クイズ              │
│                             │
│  発見済み: 12品種            │
│                             │
│  難易度を選択               │
│  ┌─────────────────────┐   │
│  │ 😺 初級              │   │
│  │ 異なる品種の見分け    │   │
│  └─────────────────────┘   │
│  ┌─────────────────────┐   │
│  │ 😼 中級              │   │
│  │ 似た特徴の品種を含む  │   │
│  └─────────────────────┘   │
│  ┌─────────────────────┐   │
│  │ 🙀 上級              │   │
│  │ 酷似品種の違いを問う  │   │
│  └─────────────────────┘   │
│                             │
│  ⚠️ 最低4品種の発見が必要    │
└─────────────────────────────┘
```

#### クイズ出題 UI
```
┌─────────────────────────────┐
│  復習クイズ      問題 2/5   │
│  ████████░░░░░░             │
│                             │
│  ┌─────────────────────┐   │
│  │  [問題文/画像]       │   │
│  └─────────────────────┘   │
│                             │
│  ○ 選択肢 A                │
│  ○ 選択肢 B                │
│  ● 選択肢 C ← 選択中       │
│  ○ 選択肢 D                │
│                             │
│        [回答する]           │
└─────────────────────────────┘
```

#### 正解/不正解フィードバック
- **正解**: 緑背景 + チェックマーク + 品種の特徴を一言表示
- **不正解**: 赤背景 + ×マーク + 正解の品種名表示

### 2. quiz-question コンポーネント (`components/quiz-question.tsx`)
- 問題文表示
- 4択の選択肢ボタン
- 選択状態の管理
- 回答確定ボタン
- 正解/不正解の色分け表示

### 3. クイズセッション管理
```typescript
// 状態
- currentQuestionIndex: number
- questions: QuizQuestion[]
- answers: (string | null)[]   // 各問題のユーザー回答
- score: number
- isAnswered: boolean          // 現在の問題に回答済みか
```

### 4. フロー制御
1. 難易度選択 → クイズ生成
2. 問題表示 → 選択 → 回答確定
3. 正解/不正解表示 → 次の問題へ
4. 全問終了 → クイズ結果画面へ遷移 (quiz-result)

### 5. 最低品種数チェック
- 発見済み品種が4未満の場合、クイズ開始不可の旨を表示
- 「図鑑で品種を発見しよう！」の案内表示

## 完了条件
- 難易度選択ができる
- 5問のクイズが出題される
- 4択の選択と回答確定ができる
- 正解/不正解が正しく判定される
- 全問終了後にクイズ結果画面へ遷移する
- 発見済み4品種未満の場合に案内が表示される
