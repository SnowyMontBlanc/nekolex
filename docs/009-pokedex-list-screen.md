# 009: 図鑑一覧画面

## 概要
発見済み品種のカード一覧、未発見品種のシルエット表示、発見進捗率を表示する図鑑画面を実装する。

## 依存先
003 (ナビゲーション), 005 (UserContext)

## ブロック対象
010, 015

## 作業内容

### 1. 図鑑画面 (`app/(tabs)/pokedex.tsx`)

#### UI 構成
```

│  🐱 図鑑 12/20              │
│  ████████░░░░░░ 60%         │
│                             │
│  [発見済み] [未発見] [すべて] │ ← フィルター
```

### 2. breed-card コンポーネント (`components/breed-card.tsx`)
- **発見済み**: 品種画像 + 名前 + 難易度表示
- **未発見**: シルエット (グレー) + 「？」
- タップで品種詳細画面へ遷移 (発見済みのみ)
- NekoLex テーマに合わせたカードデザイン

### 3. progress-bar コンポーネント (`components/progress-bar.tsx`)
- 発見品種数 / 全品種数
- パーセンテージ表示
- アニメーション付きのプログレスバー
- テーマカラー使用 (primary / secondary)

### 4. フィルター機能
- すべて / 発見済み / 未発見 の切り替え
- FlatList のグリッドレイアウト (3列)

### 5. データ取得
- UserContext から発見済み品種を取得
- breeds.json から全品種データを取得
- 発見済み/未発見を結合して表示

## 完了条件
- 全品種がグリッド表示される
- 発見済みは画像と名前が表示される
- 未発見はシルエット表示される
- 進捗バーが正しい割合を表示する
- フィルターが動作する
- 発見済み品種タップで詳細画面へ遷移する
- スクロールがスムーズに動作する
