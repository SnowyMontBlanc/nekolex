# 013: クイズ結果画面

## 概要
クイズ完了後のスコア表示、XP 獲得表示、リトライ・ホーム遷移を行う画面を実装する。

## 作業内容

### 2. データ受け渡し
クイズ画面からルートパラメータまたはグローバル状態で受け取る:
- questions: 出題された問題リスト
- answers: ユーザーの回答リスト
- difficulty: 難易度
- score: 正解数

### 3. XP 計算 & 反映
- 正解数 × 5 XP
- パーフェクト (全問正解) なら +20 XP ボーナス
- UserContext.completeQuiz() を呼び出し
- XP 付与アニメーション

### 4. 各問題の結果表示
- 正解: 品種名 + ✅
- 不正解: ユーザーの回答 + 正解の品種名

### 5. アクションボタン
- **もう一度**: クイズ画面 (難易度選択) に戻る
- **ホームへ**: ホーム画面に遷移

## 完了条件
- スコアが正しく表示される
- XP が正しく計算・付与される
- パーフェクトボーナスが適用される
- 各問題の正解/不正解が確認できる
- 「もう一度」でクイズ画面に戻れる
- 「ホームへ」でホーム画面に戻れる
