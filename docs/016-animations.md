# 016: アニメーション & 演出

## 概要
品種発見時、クイズ正解/不正解、レベルアップ時のアニメーション演出を react-native-reanimated で実装する。

## 依存先
008 (カメラ & 識別画面), 012 (クイズ画面)

## ブロック対象
なし

## 作業内容

### 1. 品種発見アニメーション
カメラ画面で新品種を発見した際の演出:
- カードフリップアニメーション
- キラキラエフェクト (パーティクル)
- 「✨ 新発見！+10 XP」テキストアニメーション
- react-native-reanimated の Animated.View + transform

### 2. クイズ正解/不正解アニメーション
- **正解**: 緑のチェックマーク + バウンスアニメーション
- **不正解**: 赤の×マーク + シェイクアニメーション
- 選択肢の色変化 (正解=緑、不正解=赤)

### 3. レベルアップアニメーション
- 画面全体に光のエフェクト
- レベル数値のカウントアップ
- 「Level Up!」テキストのスケールイン

### 4. 画面遷移アニメーション
- Stack ナビゲーションのスライドイン/アウト
- タブ切り替え時のフェード

### 5. マイクロインタラクション
- ボタンタップ時のスケール変化
- カードタップ時のリップルエフェクト
- プログレスバーのアニメーション更新

## 完了条件
- 品種発見時にフリップ + キラキラ演出が再生される
- クイズ正解/不正解で適切なアニメーションが再生される
- レベルアップ時に演出がある
- アニメーションがスムーズ (60fps) で動作する
- パフォーマンスに悪影響がない
