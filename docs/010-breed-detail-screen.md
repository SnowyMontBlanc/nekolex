# 010: 品種詳細画面

## 概要
動的ルート `breed/[id]` で品種の詳細情報 (特徴、性格、見分けポイント、似ている品種) を表示する画面を実装する。

## 依存先
009 (図鑑一覧)

## ブロック対象
なし

### 2. セクション構成
- **ヘッダー**: 品種画像 (ユーザー写真 or デフォルト)
- **基本情報**: 名前 (日/英), サイズ, 原産国, 難易度
- **説明**: description テキスト
- **特徴**: characteristics リスト
- **性格**: temperament リスト
- **見分けポイント**: distinguishing_features リスト
- **似ている品種**: similar_breeds (タップで遷移)
- **飼育上の注意**: care_notes

### 3. データ取得
- ルートパラメータ `id` から品種を特定
- breeds.json から品種データ取得
- UserContext からユーザー写真 URL 取得 (あれば表示)

### 4. 似ている品種の遷移
- similar_breeds の品種カードを表示
- タップで同画面に再遷移 (router.push)

## 完了条件
- URL パラメータから正しい品種データが表示される
- すべてのセクション (特徴、性格、見分けポイント等) が表示される
- ユーザー写真がある場合は表示される
- 似ている品種をタップすると該当品種の詳細に遷移する
- スクロール可能でレイアウトが崩れない
